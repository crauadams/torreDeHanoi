API Descrição
POST api/game/start-game/{quantidadeDeDiscos} Iniciar o jogo com uma determinada quantidade de discos.
PUT api/game/mover-disco/game-id/{gameId} Iniciar a movimentação dos discos.
GET api/log/movimento/{gameId} Obter o log das movimentações dos discos de cada jogo.
GET api/log/historico Obter o histórico de todas as execuções.
